# vw2x.github.io

My personal blog built with Next.js.

## About

This is my personal blog where I share thoughts, experiences, and technical writings.

Built with:
- Next.js App Router
- TypeScript
- Tailwind CSS
- Shadcn UI
- MDX for blog posts

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

## Deployment

Deployed on [Vercel](https://vercel.com).
